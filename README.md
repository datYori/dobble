# Dobble-like HTML5 Game

Simple pattern recognition game in which players try to find an image shown on two cards [boardgamegeek.com](https://boardgamegeek.com/boardgame/63268/spot-it).

## Features

* Vanilla JS
* Flexbox
* Iconic Font
